Listas ordenadas
Etiqueta <ol></ol>
Atributos comunes Básicos, internacionalización y eventos
Atributos propios

type = "tipo_de_marcador": Indica el tipo de marcador de la lista ordenada. El valor del tipo de marcador puede ser: "1" (números naturales, que son los que aparecen por defecto si no se pone valor), "A" (Letras mayúsculas), "a" (Letras minúculas), "I" (Números romanos mayúsculos) o "i" (Números romanos minúsculos).

start = "número": Especifica el valor del primer objeto de una lista.
Tipo de elemento Bloque
Descripción Se emplea para definir listas ordenadas

Etiqueta <li></li>
Atributos comunes Básicos, internacionalización y eventos
Atributos propios

type = "tipo_de_marcador": Indica el tipo de marcador de la lista actual. El valor del tipo de marcador puede ser: "1" (números naturales), "A" (Letras mayúsculas), "a" (Letras minúculas), "I" (Números romanos mayúsculos), "i" (Números romanos minúsculos), "disc" (Círculo con relleno), "circle" (Círculo vacío) o "square" (Cuadrado).

value = "número": Especifica el valor del objeto de la lista atual.
Tipo de elemento Bloque
Descripción Se emplea para definir los elementos de las listas (ordenadas y no ordenadas)
