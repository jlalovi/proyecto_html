Cita de texto
Etiqueta <blockquote></blockquote>
Atributos comunes Básicos, internacionalización y eventos
Atributos propios

cite = "url": Indica la URL de la página en la que se puede obtener más información sobre el motivo por el que se realizó la modificación.
Tipo de elemento Bloque
Descripción Se emplea para indicar que el texto que encierra es una cita textual de otro texto externo

Ejemplo:

Visualización

Según el W3C, el valor del atributo cite en las etiquetas blockquote tiene el siguiente significado:

"El valor de este atributo es una dirección URL que indica el documento original de la cita."

Código

<p>Según el W3C, el valor del atributo <em>cite</em> en las etiquetas <strong>blockquote</strong> tiene el siguiente significado:</p>

<blockquote cite="http://librosweb.es/xhtml/capitulo_3/marcado_basico_de_texto.html">"El valor de este atributo es una dirección URL que indica el documento original de la cita."</blockquote>

Referencia
Etiqueta <cite></cite>
Atributos comunes Básicos, internacionalización y eventos
Atributos propios -
Tipo de elemento En línea.
Descripción Se emplea para marcar una cita o una referencia a otras fuentes. En ocasiones, no está clara la diferencia entre <cite> y <blockquote>. El elemento <cite> marca el autor de la cita (persona, documento, etc.) y <blockquote> marca el contenido de la propia cita

Ejemplo:

Visualización

Como dijo Mahatma Gandhi:

Vive como si fueras a morir mañana y aprende como si fueras a vivir para siempre.

Código

<p>Como dijo <cite>Mahatma Gandhi</cite></p>

<blockquote>Vive como si fueras a morir mañana y aprende como si fueras a vivir para siempre.</blockquote>

Fuente
Etiqueta <font></font>
Atributos comunes Básicos e internacionalización
Atributos propios

color = "rgb, hexa o nombre color": Especifica el color del texto por defecto.

face = "nombre de fuente": Define el nombre de la fuente por defecto

size = "número entre 1-7": Especifica el tamaño de la fuente por defecto
Tipo de elemento En línea
Descripción Especifica el tamaño, color y tipo de fuente para un texto.

Ejemplo:

Visualización

Tamaño: 3 - color: rojo

Nombre de fuente: serif - color: azul

Nombre de fuente: verdana - color: verde

Código

<p><font size="3" color="red">Tamaño: 3 - color: rojo</font></p>

<p><font face="serif" color="blue">Nombre de fuente: serif - color: azul</font></p>

<p><font face="verdana" color="green" size="5">Nombre de fuente: verdana - color: verde</font></p>
